## How to Contribute

### Reporting Bugs
- Check if the bug has already been reported in Issues
- Include screenshots/videos if possible
- Describe your environment (OS, Python version, camera type)
- Provide steps to reproduce

### Suggesting Features
- Open an issue with the tag `enhancement`
- Describe the feature and its benefits
- Share any implementation ideas

### Code Contributions

1. **Fork & Clone**
   ```bash
   git clone https://github.com/your-username/Invisibility-Cloak-OpenCV.git
   ```

2. **Create a Branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **Make Changes**
   - Follow PEP 8 style guide
   - Add comments for complex logic
   - Test thoroughly

4. **Commit**
   ```bash
   git commit -m "Add: description of your changes"
   ```

5. **Push & PR**
   ```bash
   git push origin feature/your-feature-name
   ```

## Code Style
- Use meaningful variable names
- Add docstrings for functions
- Keep functions small and focused
- Comment non-obvious code

## Testing
Test your changes with:
- Different lighting conditions
- Various cloth colors
- Different camera resolutions
